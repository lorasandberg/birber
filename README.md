## Birber

Desktop app for managing nature photography image files.

### Main features

- Managing RAW files and preview files.
- Reframing, cropping
- Photo enhancement tools like contrast, brightness, shadows, highlights, sharpnes.
- Sort pictures by date, smart sorting into sessions and moments
- Manage burst pictures as a single unit
- Add tags and other information like species data, location, etc.
- Search and browse files by species, locations, date etc.
